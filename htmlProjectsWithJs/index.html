THIS WEBSITE ONLY TAKES IN 1-9, 10-99

Preview:
This website takes a list of numbers, adds a default value of 1 to each item, or if you can add your own number, than spits the number back out
Side Note: For some reason the algorithm taking place in the back-end will not pick up on numbers such as 100,200 etc, but if you have a number such like 401, it will delete 40 and use 1 as you input! Pretty cool huh!



RULES

1 -> If there is a zero in the "During parse", it is due to the fact that the back-end has replaced a duplicate number with zero
2 -> If there are duplicates in the list, they'll be deleted
3 -> If there is a number greater than 99 it will be deleted and replaced with 0
4 -> Make sure to put a space in between each number. Also keep note you can have as many numbers as you want, just keep it under 99!
5 -> Cannot have decimal numbers!
